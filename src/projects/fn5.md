detail:
  startingPoint: "FN5 is a conceptual clothing brand inspired by in-between spaces. Not destinations we arrive at, but spaces we pass through every day without thinking about it. These places are often optimized to be autopilot zones, which makes them memorable in a strange way. The brand exists in that same psychological space, attempting to convert this feeling into visual form."
  outcome: "FN5's visual system combines geometric elements with disruption to reflect both the invisible infrastructure and transience of in-between spaces. The Imagery uses these in-between spaces like for example elevators, where fluorescent lighting and reflective surfaces create a clinical aesthetic. The Face of the model is never fully visible, which removes personalization and emphasizes the psychological state of moving through space without full individual presence. Desaturated blue (#2f54a1) as the primary color establishes an detached feeling. Used exclusively with grays, whites, blacks, and metallics. This restrained palette mirrors the color neutrality of real in-between environments, which are often functional but not emotional. The Sans-serif typography inspired by institutional signage reinforces this logic."
